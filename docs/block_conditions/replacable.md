# Replacable

[Block Condition](../block_conditions.md).

Checks whether the block is a replacable material (e.g. tall grass, water, ...).

Type ID: `dieyourway:replacable`

### Fields:

_None._

### Example:
```json
"block_condition": {
    "type": "dieyourway:replaceable"
}
```
