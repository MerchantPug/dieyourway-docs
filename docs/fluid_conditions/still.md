# Still

[Fluid Condition](../fluid_conditions.md).

Checks whether this state describes a still (i.e. not flowing, i.e. source) fluid.

Type ID: `dieyourway:stil`

### Fields:

_None._

### Example:
```json
"fluid_condition": {
    "type": "dieyourway:still"
}
```
