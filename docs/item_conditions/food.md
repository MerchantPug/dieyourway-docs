# Food

[Item Condition](../item_conditions.md).

Checks whether the item has a food component.

Type ID: `dieyourway:food`

### Fields:

_None._

### Example
```json
"item_condition": {
    "type": "dieyourway:food"
}
```
This example checks if the item is a food item.
